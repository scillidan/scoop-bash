There are usually some non-standard package here, such as [forgit.json](bucket/forgit.json). I add it's aliases in [.alias.yaml](https://github.com/scillidan/dotfiles/blob/main/.config/alias.yaml) and use it in [clink](https://github.com/scillidan/dotfiles/blob/main/.config/_windows_clink).

## Usage

```pwsh
scoop bucket add shovel https://github.com/scillidan/scoop-shovel
```
